## 👋 Quiénes Somos
 
**Sonny S.A.C.** es una constructora de software dedicada al desarrollo de soluciones tecnológicas a la medida. Acompañamos a nuestros clientes en todo el ciclo de vida del software priorizando la **calidad, la trazabilidad y la escalabilidad** en cada entrega.
 
Este repositorio centraliza nuestros proyectos, documentación y línea base bajo un esquema ordenado de gestión de configuración.
 
---
 
## 📂 Estructura del Repositorio
 
```text
Sonny-S.A.C/
├── 📁 Clientes/         → Requerimientos, stakeholders y gestión por cuenta
├── 📁 Desarrollo/       → Código fuente, arquitectura y bases de datos
├── 📁 Documentacion/    → Políticas, planes de gestión y reportes EC
└── 📁 Linea Base/       → Hitos, entregables y trazabilidad de versiones
```
 
| Carpeta | Contenido |
| :--- | :--- |
| **`Clientes/`** | Información de proyectos desarrollados por cliente. |
| **`Desarrollo/`** | Contiene los proyectos en desarrollo de la empresa. |
| **`Documentacion/`** | Contiene los documentos transversales de la empresa. |
| **`Linea Base/`** | Contiene las líneas bases de los proyectos en desarrollo. |
